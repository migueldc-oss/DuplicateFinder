## 1. 简介

**DuplicateFinder** 是一款 macOS 应用程序，用于查找和管理磁盘上的重复文件。它可以扫描一个或多个文件夹，为每个文件计算 SHA-256 指纹，并将相同的文件分组，让您可以查看、打开、删除或导出报告。

### 主要功能

- 同时递归扫描多个文件夹。
- 两种检测模式：**可靠**（基于内容，SHA-256）和**快速**（名称 + 大小）。
- 按大小、扩展名和排除文件夹进行过滤。
- 可排序的结果表格，支持搜索。
- 单个和批量操作：打开、在 Finder 中显示、移到废纸篓或永久删除。
- 导出为 CSV 和 JSON，包含每个重复文件的位置。
- 界面支持 8 种语言。

---

## 2. 系统要求

- **macOS** 14.0（Sonoma）或更高版本。
- **Apple Silicon** 或 **Intel**。
- 无需外部软件包或依赖项。

## 3. 主界面

![界面布局](capturas/interfaz.png)

窗口分为三个区域：

| 区域 | 说明 |
|------|------|
| **顶部栏（ScanView）** | 文件夹选择、检测模式选择器、开始/取消控制。 |
| **统计栏** | 全局指标（总文件数、重复文件数、组数、可回收空间）及导出按钮。 |
| **结果表格** | 可排序的重复组列表，支持搜索。 |

---

## 4. 选择文件夹

![文件夹选择](capturas/seleccion-carpetas.png)

1. 点击 **"选择文件夹…"** 按钮（文件夹图标）。
2. 标准 macOS 对话框打开。按住 ⌘（Command）键选择**一个或多个文件夹**。
3. 文件夹会**累积**在列表中。您可以后续再次添加更多文件夹。
4. 要清除选择，点击旁边的 **"清除"** 按钮（✕ 图标）。

> **"开始扫描"** 按钮在至少选择一个文件夹之前保持禁用状态。您也可以直接从 Finder 将文件夹拖放到窗口中。

---

## 5. 检测模式

DuplicateFinder 提供两种模式，可通过顶部栏的分段控件选择：

| 模式 | 说明 |
|------|------|
| **可靠** | 计算每个文件的**完整内容**的 SHA-256 哈希值。保证无误报，但由于需要读取每个文件，速度较慢。 |
| **名称 + 大小** | 仅比较名称（不区分大小写）和字节大小。速度极快（无需读取文件），但如果不同文件具有相同的名称和大小，可能会产生误报。 |

> 默认模式在**偏好设置 → 检测**中设置。

---

## 6. 扫描进度

扫描期间，顶部栏会显示当前阶段：

| 阶段 | 指示 |
|------|------|
| **正在扫描…** | 文件夹中已处理的文件计数。 |
| **正在比较…** | 进度条，显示已计算的哈希值数量（总计候选文件数）。 |
| **正在计算…** | 数据库插入和聚合计算的进度。 |
| **完成** | 绿色勾号图标。 |
| **已取消** | 橙色图标。 |

您可以随时点击 **"取消"** 按钮取消扫描。

## 7. 结果

![结果表格](capturas/resultados.png)

扫描完成后，结果表格将显示以下列：

| 列 | 说明 |
|----|------|
| **名称** | 文件名及其图标。 |
| **大小** | 每个副本的大小。 |
| **副本数** | 该文件的重复数量。 |
| **大小 × 副本数** | 所有副本占用的总空间。 |
| **位置** | 每个副本的完整路径及操作按钮。 |

您可以点击任意列标题对表格进行**排序**。还可以使用底部的搜索字段按名称或路径**搜索**文件。

### 统计栏

结果上方显示四个指标：

- **总文件数**：所有扫描的文件。
- **重复文件**：属于重复组的文件。
- **组数**：相同文件的集合数量。
- **可回收**：删除所有重复副本（每组保留一个）后可释放的空间。

---

## 8. 文件操作

"位置"列中每个重复文件都有操作按钮：

| 按钮 | 操作 |
|------|------|
| 👁（眼睛） | 用默认应用程序打开文件。 |
| 📁（文件夹） | 在 Finder 中显示文件。 |
| 🗑（废纸篓） | 移到废纸篓（或永久删除，取决于偏好设置）。 |

### 批量删除

您可以通过勾选文件旁的方形复选框来选择多个文件：

1. 勾选要删除的文件。
2. **"删除已选 (N)"** 按钮会出现在统计栏中。
3. 点击确认并完成删除。

> 您可以在**偏好设置 → 通用**中更改删除行为：切换"移到废纸篓"选项。

---

## 9. 导出结果

点击统计栏中的 **"导出"** 按钮（分享图标），然后选择：

- **CSV** — 逗号分隔的文本文件，兼容 Excel、Numbers 和电子表格软件。
- **JSON** — 结构化格式，适合程序化处理。

两种格式均包含**每个重复文件的每一个副本**（而不只是每组一个），字段包括：名称、路径、大小、副本数、总占用空间和 SHA-256 哈希值。

将弹出 `NSSavePanel` 对话框以选择目标路径和文件名。

## 10. 偏好设置

按 `⌘,`（Command + ,）打开偏好设置，分为四个标签页。

### 通用

| 选项 | 说明 |
|------|------|
| **Top N 结果** | 结果表格中的最大行数（10–100,000）。 |
| **移到废纸篓** | 启用时，文件将被发送到废纸篓而非永久删除。 |
| **保留策略** | 自动选择保留哪个副本的策略（最旧、最新或路径最短）。 |
| **外观** | 浅色、深色或系统外观模式。 |

### 检测

| 选项 | 说明 |
|------|------|
| **默认模式** | 打开应用时默认选择的检测模式。 |

### 过滤器

| 选项 | 说明 |
|------|------|
| **最小 / 最大大小** | 大小范围（字节）。超出此范围的文件将被忽略。 |
| **包含隐藏文件** | 包含隐藏文件和文件夹（以 `.` 开头）。 |
| **跟踪符号链接** | 启用时，扫描过程中将跟踪符号链接。 |
| **允许的扩展名** | 逗号分隔的列表（例如 `jpg, png, mp4`）。空表示允许所有扩展名。 |
| **排除的文件夹** | 逗号分隔的文件夹名称列表，扫描时跳过（例如 `node_modules, .git`）。 |

### 语言

可在 8 种语言之间切换界面语言。应用可能需要重启才能完全生效。

## 11. 提示与最佳实践

1. **从快速模式开始**，如果文件夹较多。该模式速度快得多，可能已能发现大部分重复文件。
2. **删除前使用可靠模式做最终判断**——尤其是文件名不匹配时——以避免误删。
3. **选择特定文件夹**，而不是整个磁盘。仅扫描"下载"、"文稿"和"桌面"可大幅减少时间和干扰。
4. **删除前检查"位置"列**。重复文件可能位于系统文件夹中，应予以保留。
5. **导出 CSV** 再清理，以便保留已删除文件的记录。
6. **设置过滤器**来查找特定文件类型（例如，仅 `.jpg` 和 `.png` 图片）。
7. **使用"清除"按钮**重新选择文件夹，无需退出应用。
