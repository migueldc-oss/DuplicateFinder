## 1. はじめに

**DuplicateFinder** は、ディスク上の重複ファイルを検出・管理するmacOSアプリケーションです。1つ以上のフォルダをスキャンし、各ファイルのSHA-256フィンガープリントを計算して同一のものをグループ化し、確認・開封・削除・レポートのエクスポートを行うことができます。

### 主な機能

- 複数フォルダの再帰的スキャン。
- 2つの検出モード：**Reliable**（コンテンツベース、SHA-256）および **Quick**（名前＋サイズ）。
- サイズ、拡張子、除外フォルダによるフィルタリング。
- 結果内の検索と並べ替え可能なテーブル。
- 個別および一括操作：開く、Finderで表示、ゴミ箱へ移動、または完全に削除。
- すべての重複ファイルの場所を含むCSVおよびJSONへのエクスポート。
- 8言語に対応したインターフェース。

## 2. システム要件

- **macOS** 14.0（Sonoma）以降。
- **Apple Silicon** または **Intel**。
- 外部パッケージや依存関係は不要です。

## 3. メインインターフェース

![インターフェースレイアウト](capturas/interfaz.png)

ウィンドウは3つのエリアに分かれています。

| エリア | 説明 |
|--------|------|
| **上部バー（ScanView）** | フォルダ選択、検出モード選択、開始/キャンセルコントロール。 |
| **統計バー** | 全体のメトリクス（総ファイル数、重複数、グループ数、復元可能容量）とエクスポートボタン。 |
| **結果テーブル** | 検索可能な重複グループの並べ替え可能リスト。 |

## 4. フォルダの選択

![フォルダ選択](capturas/seleccion-carpetas.png)

1. **「フォルダを選択…」** ボタン（フォルダアイコン）をクリックします。
2. 標準のmacOSダイアログが開きます。⌘（Command）を押しながら**1つまたは複数のフォルダ**を選択します。
3. フォルダはリストに**追加されていきます**。後続の選択でさらにフォルダを追加できます。
4. 選択をクリアするには、横に表示される**「クリア」** ボタン（✕アイコン）をクリックします。

> **「スキャン開始」** ボタンは、少なくとも1つのフォルダが選択されるまで無効のままです。Finderからウィンドウに直接フォルダをドラッグすることもできます。

## 5. 検出モード

DuplicateFinderは、上部バーのセグメントコントロールから選択可能な2つのモードを提供します。

| モード | 説明 |
|--------|------|
| **Reliable** | 各ファイルの**全コンテンツ**のSHA-256ハッシュを計算します。誤検出がありませんが、すべてのファイルを読み込むため低速です。 |
| **Name + Size** | 名前（大文字小文字を区別しない）とサイズ（バイト単位）のみを比較します。非常に高速（ファイル読み込みなし）ですが、異なるファイルが同じ名前とサイズを持つ場合、誤検出の可能性があります。 |

> デフォルトモードは**環境設定 → Detection**で設定します。

## 6. スキャンの進行状況

スキャン中、上部バーに現在のフェーズが表示されます。

| フェーズ | インジケータ |
|----------|--------------|
| **スキャン中…** | フォルダ内で処理されたファイルのカウンター。 |
| **比較中…** | 総候補数に対する計算済みハッシュ数を示す進行状況バー。 |
| **計算中…** | データベースへの挿入と集計計算の進行状況。 |
| **完了** | 緑色のチェックマークアイコン。 |
| **キャンセル** | オレンジ色のアイコン。 |

スキャンはいつでも **「キャンセル」** ボタンをクリックして中断できます。

## 7. 結果

![結果テーブル](capturas/resultados.png)

スキャンが完了すると、結果テーブルに以下の列が表示されます。

| 列 | 説明 |
|-----|------|
| **名前** | アイコン付きのファイル名。 |
| **サイズ** | 各コピーのサイズ。 |
| **コピー数** | このファイルの重複数。 |
| **サイズ×コピー数** | 全コピーが占有する合計容量。 |
| **場所** | 各コピーのフルパスとアクションボタン。 |

任意の列ヘッダーをクリックしてテーブルを**並べ替える**ことができます。下部の検索フィールドを使用して、名前またはパスでファイルを**検索**することもできます。

### 統計バー

結果の上に4つのメトリクスが表示されます。

- **総ファイル数**：スキャンされた全ファイル。
- **重複数**：重複グループに属するファイル。
- **グループ数**：同一ファイルのセット。
- **復元可能容量**：すべての重複コピー（グループごとに1つを残す）を削除した場合に解放される容量。

---

## 8. ファイル操作

「場所」列の各重複ファイルにはアクションボタンがあります。

| ボタン | 操作 |
|--------|------|
| 👁（目） | デフォルトのアプリケーションでファイルを開く。 |
| 📁（フォルダ） | Finderでファイルを表示する。 |
| 🗑（ゴミ箱） | ゴミ箱に移動する（または環境設定に応じて完全に削除する）。 |

### 一括削除

各ファイルの横にあるチェックボックスをオンにすることで、複数のファイルを選択できます。

1. 削除したいファイルをチェックします。
2. 統計バーに **「選択したN件を削除」** ボタンが表示されます。
3. クリックして確認し、削除を完了します。

> 削除動作は**環境設定 → General**で変更できます。「ファイルをゴミ箱に移動」を切り替えてください。

## 9. 結果のエクスポート

統計バーの**「エクスポート」** ボタン（共有アイコン）をクリックして選択します。

- **CSV** — カンマ区切りのテキストファイル。Excel、Numbers、スプレッドシートと互換性があります。
- **JSON** — 構造化された形式で、プログラムによる処理に最適です。

どちらの形式も、**各重複ファイルのすべてのコピー**（グループごとに1つだけでなく）を含み、フィールドは名前、パス、サイズ、コピー数、総占有容量、SHA-256ハッシュです。

保存先とファイル名を選択するための `NSSavePanel` ダイアログが開きます。

## 10. 環境設定

`⌘,`（Command + ,）を押して環境設定を開きます。4つのタブに整理されています。

### General

| オプション | 説明 |
|------------|------|
| **Top N results** | 結果テーブルの最大行数（10～100,000）。 |
| **Move to Trash** | オンの場合、ファイルは完全に削除される代わりにゴミ箱に送られます。 |
| **Keep strategy** | 保持するコピーを自動選択する方法（最も古い、最も新しい、または最短パス）。 |
| **Appearance** | ライト、ダーク、またはシステムの外観モード。 |

### Detection

| オプション | 説明 |
|------------|------|
| **Default mode** | アプリ起動時に選択される検出モード。 |

### Filters

| オプション | 説明 |
|------------|------|
| **Min / Max size** | バイト単位のサイズ範囲。この範囲外のファイルは無視されます。 |
| **Include hidden** | 隠しファイルとフォルダ（`.` で始まるもの）を含める。 |
| **Follow symlinks** | オンの場合、スキャン中にシンボリックリンクをたどります。 |
| **Allowed extensions** | カンマ区切りリスト（例：`jpg, png, mp4`）。空の場合はすべての拡張子。 |
| **Excluded folders** | スキャン中にスキップされるフォルダ名のカンマ区切りリスト（例：`node_modules, .git`）。 |

### Language

利用可能な8言語の間でインターフェース言語を切り替えます。変更を完全に適用するにはアプリの再起動が必要な場合があります。

## 11. ヒントとベストプラクティス

1. **多くのフォルダがある場合はQuickモードから始めてください**。非常に高速で、ほとんどの重複を見つけられます。
2. **削除前の最終判断にはReliableモードを使用してください**。特にファイル名が一致しない場合に誤検出を防げます。
3. **特定のフォルダを選択し、ディスク全体は避けてください**。ダウンロード、書類、デスクトップのみをスキャンすることで、時間とノイズを大幅に削減できます。
4. **削除前に「場所」列のパスを確認してください**。システムフォルダ内に重複がある場合、そのままにしておくべきかもしれません。
5. **クリーンアップ前にCSVをエクスポートして**、削除したファイルの記録を残してください。
6. **特定のファイルタイプを探すときはフィルタを設定してください**（例：`.jpg` と `.png` 画像のみ）。
7. **「クリア」ボタンを使用して**、アプリを終了せずに新しいフォルダ選択で最初からやり直してください。
